Análise de Dados do PROUNI com Oracle SQL e Power BI
Sobre o projeto:
Projeto desenvolvido na disciplina de Laboratório de Banco de Dados com o objetivo de construir uma solução completa para armazenamento, tratamento e análise dos dados do Programa Universidade para Todos (PROUNI).
O projeto contempla a modelagem relacional do banco de dados, implementação de mecanismos de auditoria, tabelas históricas, criação de views materializadas e desenvolvimento de dashboards analíticos em Power BI.

Tecnologias utilizadas:
Oracle Database
SQL
PL/SQL
Power BI
Data Warehouse
ETL
Funcionalidades
Modelagem relacional baseada nos dados do PROUNI
Processo ETL para tratamento dos dados
Controle histórico de alterações
Auditoria através de triggers
View materializada para otimização de consultas
Dashboard analítico focado nos beneficiários
